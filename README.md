# gestorConveniencia

**Criação de um sistema web para gerenciamento de estoque e vendas de loja de conveniência**

**Última atualização**: 2017

**Tecnologias utilizadas**:
- JEE 7
- JSF 2.2
- JPA 2.1
- CDI - Jboss Weld
- Java Bean Validations
- Maven
- PostgresSQL
- Tomcat Server
- Template Twitter Bootstrap
